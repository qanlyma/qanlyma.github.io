本文最后更新于：1 个月前

Markdown 是一种轻量级标记语言，在 2004 年由 John Gruber 创建。它允许人们使用易读易写的纯文本格式编写文档。
本文介绍使用 Markdown 编写博客时常用的标记符号供后续使用时参考。

Markdown 标题

使用 # 号标记，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

Markdown 格式

可以实现的格式： 斜体文本 ， 粗体文本 ， 粗斜体文本 ， 删除线 ， 带下划线文本
